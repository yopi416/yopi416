## 自己PR

### 強み
① 高い目標意識をもって努力できる
- 必要なことは、業務外や休日でも学習します
- 例1: 当初は、元請けとしてベンダマネジメントを行う場面が多く、ネットワーク／サーバ／クラウドなど幅広い技術理解が必要だった<br>
       ⇒ 1年目のうちに関連資格を一通り取得してキャッチアップ（上記資格欄参照)
- 例2: （仕事とは関係ないですが）ゴルフを始めて1年以内にスコア100切りすることを目標に設定<br>
       ⇒ 最後の一か月間は、週4,5の頻度で22:00-4:00まで練習を行い、無事目標達成
  
② 自走力が高い<br>
- 現職ではほとんどのプロジェクトでリーダーを務めてきたため、必要なタスクを自ら拾い、主体的に進めることができます
- タスクを任される際も、100のうち1だけ言っていただければ、残りは自分で情報収集し、実行まで進められます
- 運動会の団幹部や文化祭の企画経験など、学生時代からまとめ役になることが多かったため、自ら動く習慣が身についたと思っています
<br>

③ 人間関係を円滑にしたままPJを推進できる<br>
- 感謝や気遣いを意識し、コミュニケーションの摩擦を減らすことを大切にしています。
- 実際に「あなたのプロジェクトはやりやすい」と人づてに伺うこともありました
<br>
<br>


### 他者と仕事をする上で大事にしていること
① 対立構造を作らない・喧嘩腰にならないこと<br>
- 対立構造が生まれた瞬間に建設的な議論ができなくなると思っています
- そのため、例え相手が100%悪いと思えるような場合であっても、喧嘩腰にならず、腰を低く話すようにしています
- 学生時代に、強い言い方で意見をぶつけてしまったことで、大きな失敗をした経験があり、そこから伝え方を意識するようになりました
<br>

② おかしいと思ったことは役職に関係なく伝えること<br>
※ 本ポイントはまだ100%できているわけではないですが、意識はしています<br>
- 経験上、自分が違和感を持つときは、周囲の方も同じ考えを抱いている可能性が高いです
- また、管理職の方でも、実務担当よりも情報が少ないことが原因で誤った方向に進むことがあるため、適切なコメントはむしろ感謝されることが多いと感じています
- ただし、伝え方を誤ると話がこじれる可能性があるため、相手への配慮を忘れず、丁寧に伝えることを心がけています
<br>

③ 相手への配慮を忘れないこと<br>
- 小さなことかもしれませんが、例えば以下のようなことを意識しています
- 例1: 何か自分にしていただいたら必ず感謝を伝える
- 例2: チャットで他者に意見を求める際に、「意見がなければスタンプだけで大丈夫です」と添える
　　  ⇒相手が余計な文章を考えたり、長い返信を作る負担を減らすことができる
